# Travelmap
Travel data storage

# How to contribute

## Requirements

This project is guaranteed to work with following configuration: 

- Ubuntu 24.04.3 LTS 
- Python 3.12.3
- Pip 24.0
- Git 2.43.0
- psql (PostgreSQL) 16.11 (Ubuntu 16.11-0ubuntu0.24.04.1) 

## Local run

- Install required packages
- Clone this repo
- Create virtual env if it does not exist(`python3 -m venv venv`)
- Run virtual env (`source venv/bin/activate`)
- Apply requirements (`pip3 install -r requirements.txt`)
